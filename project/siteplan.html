WDD 131 – Project Site Plan

Site Name

Electrician Career Guide

This name was chosen because the website focuses on providing clear, practical information about becoming and working as an electrician. The name is simple, professional, and directly reflects the purpose of the site.

Site Purpose

The purpose of this website is to inform users about the electrician trade. The site will provide information about job responsibilities, required skills, common tools, apprenticeship and training paths, workplace safety, and long-term career growth opportunities.

Scenarios

  • What steps do I need to take to become an electrician?
  • What tools and skills are required for entry-level electricians?

Color Scheme

The following colors will be used throughout the site:

  • #1f2933 (Dark Blue-Gray): Used for headers, navigation bar, and footer.
  • #fbbf24 (Safety Yellow): Used for accents, buttons, and highlights.
  • #ffffff (White): Used for main backgrounds and readability.

Typography

  • Roboto: Used for body text and general content.
  • Montserrat: Used for headings and navigation.
